# react_memory_game
A memory game using React.

## Deployed
 - [via Heroku](https://click-memory.herokuapp.com/)

## Built With

* React
* Node.js

### Prerequisites

- [Node.js](https://nodejs.org/en/)
- [npm](https://www.npmjs.com)

### Installing

1. Clone this repo down to your local machine and cd into it
2. `cd react_memory`
2. `npm install`

### Running

1. `npm start`
