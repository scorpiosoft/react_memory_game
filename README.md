# react_memory_game
A memory game using React.

## Deployed
 - [via Heroku](https://click-memory.herokuapp.com/)

## Built With

* React
* Node.js

### Prerequisites

- [Node.js](https://nodejs.org/en/)
- [npm](https://www.npmjs.com)
- [yarn](https://www.npmjs.com/package/yarn)

### Installing

1. Clone this repo down to your local machine
2. `yarn install`

### Running

1. `yarn start`
